Matlab code for Regression for Informative Projection Recovery (RIPR)

References:
http://www.cs.cmu.edu/~mfiterau/papers/2012/recip_nips2012.pdf
http://www.cs.cmu.edu/~mfiterau/papers/2013/ipr_icmla.pdf
http://www.cs.cmu.edu/~mfiterau/papers/2015/active_ripr_aaai.pdf


Getting started:

1. Install the cvx package
http://cvxr.com/cvx/

2. Download VIPR from githib and add all its subfolders to the path

3. Run the script VIPR/exp/reg_uci_concrete/RunVIPR.m
